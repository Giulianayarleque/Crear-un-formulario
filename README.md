# 🍋 Little Lemon - Formulario de Reservas

Proyecto web de un formulario de reservas para **Little Lemon**, un restaurante mediterráneo familiar que combina recetas tradicionales con un toque moderno.

## 📋 Descripción

Este proyecto consiste en una página web estática que permite a los clientes realizar una reserva en el restaurante ingresando sus datos básicos.

## 🌐 Vista previa

> Puedes abrir el archivo `index.html` directamente en tu navegador.

## 🧾 Campos del formulario

- **Email** – Dirección de correo del cliente
- **Fecha de reserva** – Selector de fecha
- **Número de personas** – Entre 1 y 8 personas
- **Política de cancelación** – Checkbox de aceptación obligatoria
- **Botón "Book Now"** – Envía la reserva

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
